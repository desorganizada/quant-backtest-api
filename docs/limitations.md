Execution Scope

This repository is published as a portfolio project and does not represent a fully self-contained external application.

Some parts of the original workflow depend on:

private data sources
restricted execution environments
external permissions
an external backtesting package
Reproducibility

Because of these dependencies, the complete original workflow may not be reproducible outside its intended environment.

The repository is therefore presented primarily as a demonstration of:

code organization
ETL and parsing workflow design
backend architecture
integration structure for quantitative systems
Sensitive Artifacts

Original execution outputs, raw logs, and internal analytical artifacts were reduced or excluded from the public version of the project to avoid exposing sensitive or proprietary information.

Only simplified and sanitized examples are included in the repository.
